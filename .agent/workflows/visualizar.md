# Instruções

O usuário quer visualizar a página no navegador.

## Sua Tarefa

1. Verifique se o servidor Netlify Dev está rodando
2. Se não estiver, inicie com `netlify dev`
3. Forneça o link clicável: http://localhost:8888

## Comando

```bash
netlify dev
```

O servidor roda na porta 8888 por padrão.

## Importante

- Se o servidor já estiver rodando, apenas forneça o link
- O servidor precisa estar rodando para o CDN de imagens funcionar (`/.netlify/images`)
- Se houver erro, verifique se está na pasta correta do projeto
